# GUI 使用指南

## 启动方式

```bash
streamlit run src/gui/app.py
```

或：

```bash
python scripts/run_gui.py
```

## 功能说明

1. 侧边栏可直接触发全流程运行。
2. 支持切换是否启用真实 LLM 调用。
3. 主界面显示人模 Spearman 与系统五维均值。
4. 可分页预览 Fig1-Fig4，并下载对应 PDF。
5. 可查看评测集/翻译样本及实验日志/方法文档内容。

## 常见问题

- 如果页面提示产物不存在，请先点击“运行全流程”。
- 如未配置 API Key，建议先关闭“启用真实 LLM 调用”。
- 若图表中文显示异常，请确认系统安装中文字体（微软雅黑/黑体）。
